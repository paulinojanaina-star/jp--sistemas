# JP Sistemas - Controle de Estoque

Sistema interno para controle de estoque de unidades de saúde com alertas de reposição e histórico de movimentações.

## Visão Geral
Este aplicativo permite gerenciar inventário hospitalar de forma eficiente. O sistema previne a saída de itens sem saldo, emite alertas de estoque baixo e provê um histórico detalhado de operações realizadas por cada responsável.

## Tecnologias Utilizadas
- React 19
- Vite
- TailwindCSS
- shadcn/ui
- Supabase (Autenticação e Banco de Dados com RLS)
